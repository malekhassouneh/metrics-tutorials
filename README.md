# metrics-tutorials

A repository containing tutorial materials for an upper-year undergraduate introductory econometrics course. Tutorials focused on implementing econometric analysis in statistical software (Stata + R). Below are more details on the course for which these materials were developed.

**University:** University of Toronto, Department of Economics \
**Course:** ECO 375 - Applied Econometrics I (Fall 2022 - Spring 2023) \
**Instructors:** Eduardo Souza-Rodrigues (Fall 2022), Jiaying Gu (Spring 2023) \
**Reference:** Introductory Econometrics: A Modern Approach 7e (Wooldridge 2020) [Ch.1-8 & 15]

### File Overview

Below is a high-level overview of the contents of each tutorial file. Each file corresponds to a weekly tutorial session, including annotated code and examples carried out in statistical software (Stata and R). A description of the content covered and the corresponding chapters in the reference material (Wooldridge 2020) are also provided. Note that the end and start of consecutive files can overlap in content.
